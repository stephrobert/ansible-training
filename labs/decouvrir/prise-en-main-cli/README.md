# Lab decouvrir/prise-en-main-cli

Repérage des **8 commandes Ansible du quotidien**. Ce lab est purement **déclaratif** : aucun playbook à exécuter, juste des commandes à taper depuis le control node sur le lab existant (`inventory/hosts.yml`).

## Lancer le tour

```bash
make run
```

Affiche en série une commande type pour chaque outil et la sortie obtenue : `ansible`, `ansible-playbook`, `ansible-doc`, `ansible-galaxy`, `ansible-vault`, `ansible-config`, `ansible-inventory`, `ansible-lint`.
